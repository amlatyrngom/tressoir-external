# Canon Artifacts

This directory contains reviewed, validated files that are known-good reusable
examples for this project, such as configuration files, scripts, templates,
fixtures, and small reference implementations.

Canon artifacts are part of project canon and should be version-controlled.
For each example, document its purpose, usage, prerequisites, validation, and
important limits here or in a nearby Markdown file. Use descriptive stable
paths, and keep secrets, machine-specific state, generated output, dependencies,
binaries, raw downloads, active task work, and unreviewed experiments elsewhere.
